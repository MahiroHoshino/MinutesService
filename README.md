# MinutesService

## ファイル構成

- root
    - static (データ関係)
        - download (ダウンロードデータ一時保存場所)

    - server.py (サーバー flask)

## 使用言語
- サーバーサイド
    - python(Flask)
- フロントエンド
    - html
    - javascript (jQuery)
